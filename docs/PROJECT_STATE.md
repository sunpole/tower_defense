# Текущее состояние проекта

## Статус

Проект является рабочим браузерным MVP Tower Defense.

Игровой цикл подтверждён локальной проверкой:

1. игрок выбирает башню;
2. размещает её на свободной клетке;
3. запускает волну;
4. башни автоматически атакуют врагов;
5. игрок получает энергию за уничтожение;
6. после пяти волн наступает победа;
7. при потере здоровья базы наступает поражение.

## Технологии

- React 19;
- TypeScript 5.9;
- Vite 8;
- ESLint 9;
- CSS без отдельной UI-библиотеки.

## Архитектура

- `components/` — отображение и действия интерфейса;
- `game/` — чистая игровая логика и геометрия;
- `hooks/` — жизненный цикл игрового таймера;
- `config/` — параметры поля, башен и врагов;
- `types/` — общие TypeScript-типы.

## Подтверждённые проверки

```bash
npm run build
npm run lint
```

Обе команды должны завершаться без ошибок.

## Ограничения текущей версии

- только один тип врага;
- красная и синяя башни пока отличаются цифрами, но не визуальной механикой атаки;
- нет выбора установленной башни;
- нет улучшения и продажи;
- нет паузы и изменения скорости;
- нет сохранения прогресса;
- нет звука и окончательной графики;
- пока не подключены 15 городов, благословения и валюта «ВЕРА».

## Важное правило разработки

Каждый новый этап выполняется в отдельной ветке и объединяется с `main` только после успешных локальных проверок и ручной проверки игры.
