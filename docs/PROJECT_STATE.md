# Текущее состояние проекта

## Статус

Проект является рабочим браузерным MVP Tower Defense.

Текущий этап: `v0.7.0 — Кубическая судьба`, черновой PR №9.

## Игровой цикл

1. Получить случайный одноклеточный маршрут.
2. Покупать простые башни за энергию.
3. Защищать базу на протяжении 30 волн.
4. Уничтожать скрытых Кубиков судьбы и получать `🎲`.
5. Выполнять случайные слияния башен одного уровня.
6. Уничтожать боссов и получать `🧬`.
7. Вручную выбирать две башни одного уровня для суперслияния.
8. Создавать башни более высокого уровня.

## Технологии

- React 19;
- TypeScript 5.9;
- Vite 8;
- ESLint 9;
- CSS без UI-библиотеки;
- GitHub Actions.

## Реализовано

- полноэкранный трёхколоночный интерфейс;
- компактный магазин и безопасная продажа;
- случайные маршруты с seed;
- 30 волн;
- четыре обычных архетипа, кубики, мини-боссы и боссы;
- 27 двухцветных результатов;
- случайное слияние за `🎲`;
- суперслияние за `🧬`;
- уровни башен с шагом усиления 50%;
- HP под каждым врагом;
- минимальные CSS-эффекты;
- пауза, ускорение и автозапуск;
- debug-режим;
- итоговый отчёт и локальная статистика;
- автоматическая диагностика 500 маршрутов и распределения кубиков;
- Chrome-лаунчер с обычной и debug-вкладкой.
