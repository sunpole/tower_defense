# v0.5.0 — Цветовой Завет

## Идея

Башни больше не развиваются обычной кнопкой улучшения. Сила появляется только через слияние цветов.

## Базовые цвета

- Красный — радиус, фокус и добивание.
- Зелёный — перезарядка, темп и повторные атаки.
- Синий — распространение, рикошет и цепные атаки.

## Правило состава

Сумма цвета гибрида всегда равна 10.

Примеры:

- Красный 5 + Зелёный 5;
- Красный 7 + Синий 3;
- Зелёный 1 + Синий 9.

## Вероятности первого слияния

| Состав | Вероятность |
|---|---:|
| 5/5 | 70% |
| 6/4 | 7.5% |
| 4/6 | 7.5% |
| 7/3 | 5% |
| 3/7 | 5% |
| 8/2 | 2% |
| 2/8 | 2% |
| 9/1 | 0.5% |
| 1/9 | 0.5% |

## Главное правило способностей

- 5/5 получает две способности — по одной от каждого цвета.
- 6/4, 7/3, 8/2 и 9/1 получают только способность доминирующего цвета.
- Меньший цвет продолжает влиять на основную характеристику, но не открывает свою способность.
- Способности предыдущих ступеней не складываются: 9/1 получает способность ступени 9, а не способности 5–9 одновременно.

## Стоимость слияния

Базовая цена первого слияния чистых башен — 90 энергии.

Чем реже башня и выше её ранг, тем дороже следующее слияние.

| Редкость | Множитель следующего слияния |
|---|---:|
| Равновесная | ×1.0 |
| Необычная | ×1.25 |
| Редкая | ×1.6 |
| Эпическая | ×2.2 |
| Легендарная | ×3.0 |

Дополнительно применяется множитель ранга: каждый следующий ранг повышает цену.

## Как выполнить слияние

1. Нажать на установленную башню.
2. В панели увидеть совместимые башни и диапазон цен.
3. Нажать «Начать слияние».
4. На поле совместимые башни начнут пульсировать.
5. Рядом с каждой совместимой башней появится точная цена.
6. Зелёная цена означает, что энергии хватает.
7. Красная цена означает, что сначала нужно накопить энергию.
8. Нажать на выбранную совместимую башню.

## Атлас слияний

Атлас открывается в отдельной боковой панели и не увеличивает высоту страницы.

Для каждого из 27 результатов он показывает:

- название;
- состав;
- вероятность;
- редкость;
- урон;
- радиус;
- перезарядку;
- количество целей;
- полное описание способности;
- точные числовые эффекты.

Числа атласа рассчитаны для первого слияния двух чистых башен, то есть для ранга 2.

## Количество волн

В цветовой версии используется 10 волн. Пять волн оказалось недостаточно, чтобы игрок успевал строить несколько последовательных слияний и почувствовать ценность редких результатов.

## Ограничение текущего этапа

В v0.5.0 реализованы только двухцветные слияния. Три цвета будут проектироваться отдельно после проверки и балансировки 27 двухцветных результатов.
