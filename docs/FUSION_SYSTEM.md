# Система цветовых слияний

## Главный принцип

Башни развиваются только через слияние цветов. Обычной кнопки улучшения нет.

## Базовые цвета

- **Красный** — радиус, фокус, добивание.
- **Зелёный** — перезарядка, темп, повторные атаки.
- **Синий** — распространение, рикошет, дополнительные цели.

## Состав

Сумма двух цветов всегда равна 10.

Возможные результаты первого слияния:

| Состав | Вероятность | Редкость |
|---|---:|---|
| 5/5 | 70% | Равновесная |
| 6/4 | 7,5% | Необычная |
| 4/6 | 7,5% | Необычная |
| 7/3 | 5% | Редкая |
| 3/7 | 5% | Редкая |
| 8/2 | 2% | Эпическая |
| 2/8 | 2% | Эпическая |
| 9/1 | 0,5% | Легендарная |
| 1/9 | 0,5% | Легендарная |

## Способности

- 5/5 активирует две способности — по одной от каждого цвета;
- 6/4–9/1 активируют только способность доминирующего цвета;
- меньший цвет продолжает влиять на итоговые характеристики;
- способности предыдущих ступеней не складываются;
- точные числа каждой способности показаны в атласе.

## Последовательное слияние

Гибрид можно соединить с чистой башней одного из уже присутствующих цветов.

Пример:

```text
Красный 5 + Зелёный 5
+ чистая Красная
= Красный 6 + Зелёный 4
```

Каждое последовательное слияние:

- повышает ранг;
- смещает состав на одну единицу;
- пересчитывает урон, радиус, перезарядку и количество целей;
- может заменить активную способность;
- увеличивает стоимость дальнейшего слияния.

## Цена

Базовая цена: 90 энергии.

Множители редкости:

| Редкость | Множитель |
|---|---:|
| Равновесная | ×1,0 |
| Необычная | ×1,25 |
| Редкая | ×1,6 |
| Эпическая | ×2,2 |
| Легендарная | ×3,0 |

Также цена растёт вместе с рангом участвующих башен.

## Интерфейс

1. Выбрать установленную башню.
2. Увидеть совместимые цели и диапазон цен.
3. Нажать «Начать слияние».
4. Выбрать подсвеченную башню.

- зелёный ценник — энергии хватает;
- красный ценник — энергии недостаточно;
- при нехватке показывается точная недостающая сумма;
- атлас открывается в отдельной боковой панели и не увеличивает высоту страницы.

## Атлас

Для каждого из 27 результатов показываются:

- название;
- состав;
- вероятность;
- редкость;
- урон;
- радиус;
- перезарядка;
- количество целей;
- полное описание способности;
- точные числовые эффекты.

Числа атласа рассчитаны для первого слияния двух чистых башен — ранг 2.

## Визуальная обратная связь

- успешное слияние создаёт короткий цветовой импульс;
- особая способность подписывается только в момент реального срабатывания;
- красные, синие и зелёные атаки имеют разные минимальные эффекты;
- эффекты не должны перекрывать поле или превращаться в постоянный визуальный шум.

## Количество волн

Цветовая версия использует 10 волн. Это даёт время построить несколько последовательных слияний и почувствовать ценность редких результатов.

## Ограничение v0.5.0

Реализованы только двухцветные сочетания. Трёхцветная система будет проектироваться после проверки баланса 27 текущих результатов.
