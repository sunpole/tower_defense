# Правила текущей версии

## Цель

Не дать врагам пройти по дороге до базы и пережить пять волн.

## Начальные параметры

- энергия: 180;
- здоровье базы: 10;
- количество волн: 5;
- стоимость установки любой тестовой башни: 50 энергии.

## Размещение башен

- башню можно поставить только на свободную клетку вне дороги;
- на одной клетке может находиться только одна башня;
- после установки стоимость вычитается из энергии;
- во время волны новые башни устанавливать разрешено;
- только что установленная башня автоматически становится выбранной.

## Управление установленной башней

- башня выбирается нажатием на неё на игровом поле;
- выбранная башня подсвечивается, а на поле показывается её радиус атаки;
- справа отображаются уровень, урон, радиус, перезарядка и вся вложенная энергия;
- максимальный уровень башни: 5;
- при улучшении повышаются урон и радиус, а перезарядка уменьшается;
- стоимость улучшений растёт с каждым уровнем;
- продать башню можно в любой момент;
- при продаже возвращается 60% всей энергии, вложенной в установку и улучшения.

## Типы башен

### Красная

- одиночная цель;
- урон: 15;
- радиус: 2 клетки;
- перезарядка: 800 мс.

### Синяя

- одиночная цель;
- урон: 20;
- радиус: 3 клетки;
- перезарядка: 600 мс.

### Зелёная

- атакует всех врагов в радиусе;
- урон: 10;
- радиус: 2 клетки;
- перезарядка: 1200 мс.

## Волны

- первая волна содержит 5 врагов;
- каждая следующая волна содержит на 2 врага больше;
- с каждой волной растут здоровье и скорость врагов;
- после завершения волн 1–4 начисляется бонус 25 энергии.

## Бой

- одиночные башни атакуют врага, который дальше всех прошёл по маршруту;
- зелёная башня наносит урон всем врагам в своём радиусе;
- за уничтоженного врага начисляется энергия;
- каждый враг, дошедший до конца маршрута, отнимает 1 здоровье базы.

## Победа и поражение

- победа: уничтожить всех врагов пятой волны;
- поражение: здоровье базы становится равным нулю;
- кнопка «Начать заново» полностью сбрасывает бой, сохраняя выбранный тип башни.
