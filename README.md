# Tower Defense

Браузерная Tower Defense игра на React, TypeScript и Vite.

Проект находится в активной разработке. Текущая версия содержит первый полностью проходимый игровой цикл: размещение башен, пять волн врагов, энергию, здоровье базы, победу, поражение и перезапуск.

## Что уже работает

- поле 12×8 с фиксированным маршрутом;
- три тестовые башни: снаряд, лазер и аура;
- размещение башен вне дороги;
- расход и получение энергии;
- автоматический выбор целей и атака;
- пять волн с ростом здоровья и скорости врагов;
- здоровье базы;
- победа, поражение и повторный запуск игры;
- адаптивный интерфейс;
- автоматические проверки сборки и ESLint в GitHub Actions.

## Запуск

Требуется Node.js 22 и npm.

```bash
npm ci
npm run dev
```

После запуска откройте адрес, который покажет Vite, обычно:

```text
http://localhost:5173/
```

## Проверка проекта

```bash
npm run build
npm run lint
```

## Основная структура

```text
src/
├── components/   интерфейс игры
├── config/       башни, враги и настройки поля
├── game/         правила боя, движение и создание врагов
├── hooks/        игровой таймер
└── types/        TypeScript-типы
```
