# Tower Defense — «Цветовой Завет»

Браузерный прототип Tower Defense на React, TypeScript и Vite.

Главная особенность проекта — башни развиваются не обычной кнопкой улучшения, а через слияние красного, зелёного и синего цветов. Цветовой состав определяет характеристики, редкость, название и способности башни.

Версия `v0.6.0 — Живой маршрут` добавляет случайную карту каждой новой партии, 30 волн, боссов и встроенный полигон балансировки.

## Текущий игровой цикл

1. Получить новую карту со своим seed.
2. Изучить длину, повороты и поправку сложности маршрута.
3. Поставить чистые башни рядом с дорогой.
4. Посмотреть цену и вероятности слияния.
5. Создать цветовые гибриды.
6. Пережить 30 постепенно усложняющихся волн.
7. Победить мини-боссов на 5, 15 и 25 волнах.
8. Победить больших боссов на 10, 20 и 30 волнах.

## Живой маршрут

- новая дорога генерируется при старте и по кнопке «Новая карта»;
- путь начинается слева и заканчивается справа;
- клетки пути не повторяются и дорога не пересекает себя;
- дорога всегда остаётся шириной в одну клетку;
- маршрут остаётся неизменным до конца партии;
- игрок видит seed, длину, число поворотов и поправку HP;
- длинный путь умеренно усиливает врагов и немного повышает награду.

## Цвета

- красный — радиус, фокус и добивание;
- зелёный — скорость перезарядки, серии и импульс по всем врагам в радиусе;
- синий — распространение, рикошет и дополнительные цели.

Доступны 27 первых двухцветных результатов: от равновесных 5/5 до легендарных 9/1.

## Волны и боссы

Состав каждой волны рассчитывается заранее и показывается игроку до запуска.

- 30 волн;
- обычные, быстрые, тяжёлые и элитные враги;
- мини-босс на волнах 5, 15 и 25;
- большой босс на волнах 10, 20 и 30;
- отдельная полоса здоровья босса;
- босс визуально не превышает размер клетки;
- мини-босс наносит базе 3 урона, большой босс — 5.

## Полигон баланса

В обычной игре постоянно доступны:

- пауза;
- скорости ×1, ×2 и ×4;
- автозапуск следующей волны;
- итоговый отчёт после победы или поражения;
- локальная статистика побед, поражений и средней достигнутой волны.

Статистика хранится только в `localStorage` браузера и никуда не отправляется.

Скрытый режим разработчика открывается по адресу:

```text
http://localhost:5173/?debug=1
```

В нём доступны быстрые переходы к волнам 5, 10, 15, 20, 25 и 30, добавление энергии, восстановление базы, повтор seed-карты и автоматическая проверка 500 маршрутов.

## Визуальные эффекты

Снаряды, лазеры, ауры и срабатывания способностей показаны короткими CSS-анимациями. Эффекты намеренно минимальны и не используют тяжёлые графические библиотеки.

## Запуск

Требуется Node.js 22 и npm.

```bash
npm ci
npm run dev
```

Откройте адрес, который покажет Vite, обычно:

```text
http://localhost:5173/
```

## Проверки

```bash
npm run build
npm run lint
```

GitHub Actions выполняет эти проверки автоматически для Pull Request.
