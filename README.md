# Tower Defense — «Цветовой Завет»

Браузерная Tower Defense на React, TypeScript и Vite.

Версия `v0.7.0 — Кубическая судьба` строится вокруг трёх систем:

- случайный одноклеточный маршрут каждой партии;
- цветовые башни и 27 двухцветных сочетаний;
- две валюты слияния, которые добываются в бою.

## Игровой цикл

1. Получить карту со своим seed.
2. Покупать красные, синие и зелёные башни за энергию.
3. Защищать базу на протяжении 30 волн.
4. Искать скрытых белых Кубиков судьбы.
5. Тратить `🎲` на случайное слияние двух башен одного уровня.
6. Уничтожать боссов и получать `🧬`.
7. Выполнять суперслияние двух выбранных башен одного уровня.
8. Создавать башни всё более высокого уровня.

## Экономика

### Энергия

Используется только для покупки простых башен. Слияния больше не требуют энергии.

### Очки слияния `🎲`

Кубик судьбы приносит столько очков, сколько точек выпало на его грани.

Одно очко:

- случайно выбирает две башни одного уровня;
- создаёт случайную двухцветную башню;
- использует вероятности от 70% для 5/5 до 0,5% для конкретного 9/1;
- не повышает уровень результата.

### Очки суперслияния `🧬`

Уничтоженный мини-босс или большой босс даёт одно очко.

Одно очко позволяет вручную выбрать две башни одного уровня. Результат получает следующий уровень, а каждый новый уровень добавляет 50% к базовым характеристикам.

## Кубики судьбы

В каждом блоке из пяти волн скрыто распределяются 3–10 кубиков.

- цвет — белый;
- грань — от `⚀` до `⚅`;
- высокая грань означает больше HP и скорости;
- кубик не наносит урон базе;
- не даёт обычную энергию;
- количество и волна появления заранее не показываются;
- тот же seed воспроизводит то же распределение.

## Маршрут и волны

- новая одноклеточная дорога создаётся при старте партии;
- путь не пересекается и не касается сам себя боком;
- длина маршрута умеренно влияет на HP и награду;
- всего 30 волн;
- мини-боссы: 5, 15, 25;
- большие боссы: 10, 20, 30.

## Интерфейс

- постоянный атлас слева;
- игровое поле по центру;
- компактный магазин и синтез справа;
- HP подписан под каждым врагом;
- жёлтая перезарядка расположена под башней;
- допустимые цели суперслияния отмечаются `🧬`;
- продажа требует отдельного подтверждения.

## Полигон баланса

Доступны пауза, скорости ×1/×2/×4, автозапуск волн, итоговый отчёт и локальная статистика.

Debug-режим:

```text
http://localhost:5173/?debug=1
```

Он позволяет переходить к боссовым волнам, добавлять энергию, `🎲` и `🧬`, восстанавливать базу и проверять 500 seed-карт.

## Запуск

Требуется Node.js 22 и npm.

```bash
npm ci
npm run dev
```

`npm run dev` запускает Vite и открывает Google Chrome с двумя вкладками:

```text
http://localhost:5173/
http://localhost:5173/?debug=1
```

Запуск без автоматического Chrome:

```bash
npm run dev:vite
```
