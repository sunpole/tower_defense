# Tower Defense — «Цветовой Завет»

Браузерный прототип Tower Defense на React, TypeScript и Vite.

Главная особенность проекта — башни развиваются не обычной кнопкой улучшения, а через слияние красного, зелёного и синего цветов. Цветовой состав определяет характеристики, редкость, название и способности башни.

## Текущий игровой цикл

1. Выбрать чистую башню.
2. Поставить её рядом с дорогой.
3. Подготовить несколько совместимых башен.
4. Посмотреть цену и вероятности слияния.
5. Создать цветовой гибрид.
6. Пережить 10 постепенно усложняющихся волн.

## Цвета

- красный — радиус, фокус и добивание;
- зелёный — скорость перезарядки, серии и повторные атаки;
- синий — распространение, рикошет и дополнительные цели.

Доступны 27 первых двухцветных результатов: от равновесных 5/5 до легендарных 9/1.

## Баланс

Волны детерминированы и заранее показываются игроку. Сложность растёт через:

- количество врагов;
- здоровье;
- скорость;
- появление быстрых, тяжёлых и элитных противников;
- более медленный рост награды относительно угрозы.

Текущий ориентир — около 70% побед и 30% поражений после освоения механики слияния. Для точной настройки нужны повторные игровые тесты.

## Визуальные эффекты

Снаряды, лазеры, ауры и срабатывания способностей показаны короткими CSS-анимациями. Эффекты намеренно минимальны и не используют тяжёлые графические библиотеки.

## Запуск

Требуется Node.js 22 и npm.

```bash
npm ci
npm run dev
```

Откройте адрес, который покажет Vite, обычно:

```text
http://localhost:5173/
```

## Проверки

```bash
npm run build
npm run lint
```

GitHub Actions выполняет эти проверки автоматически для Pull Request.

## Основная структура

```text

├── components/   интерфейс игры
├── config/       башни, враги и настройки поля
├── game/         бой, слияния, волны и баланс
├── hooks/        игровой таймер
└── types/        TypeScript-типы
```
