# v0.6.1 — Компактный интерфейс

## Главное

Этот патч не добавляет новую игровую механику. Он приводит уже существующие панели к единой визуальной системе и освобождает место для самой игры.

## Магазин

- три базовые башни теперь показаны одновременно в одной компактной сетке;
- убраны повторяющиеся длинные описания из постоянного вида;
- сохранены цвет, тип атаки, урон, радиус, цена и состояние доступности;
- подробное описание остаётся в подсказке при наведении;
- недоступные покупки по-прежнему блокируются.

## Башня и слияние

- характеристики собраны в плотную сетку;
- цветовой состав показан компактными цветными метками;
- способности больше не растягивают панель лишними отступами;
- шаги слияния собраны в короткую шкалу `Башня → Режим → Цель`;
- цена, запас энергии и число доступных целей находятся рядом;
- основная кнопка слияния занимает только необходимую высоту;
- на экране 1920×1080 панель рассчитана на работу без внутренней прокрутки.

## Безопасная продажа

- продажа вынесена из основного ряда действий;
- первый клик только открывает подтверждение;
- подтверждение показывает точную сумму возврата;
- доступны отдельные кнопки отмены и подтверждения;
- случайная продажа одним нажатием больше невозможна.

## Единая UI-система

Добавлен документ `docs/UI_SYSTEM.md` и CSS-токены для:

- отступов;
- радиусов;
- высоты кнопок;
- плотности карточек;
- опасных действий;
- правил добавления новых постоянных панелей.

## Ограничения

Патч не меняет:

- баланс;
- урон;
- маршруты;
- волны;
- боссов;
- механику слияния;
- локальную статистику.

Изменяется только представление и безопасность управления.
