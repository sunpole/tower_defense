# v0.7.0 — Кубическая судьба

## Слоган

**Белые кубики скрываются в волнах. Боссы открывают путь к настоящему росту башен.**

## Главная перемена

Старое ручное слияние за энергию удалено. Энергия теперь используется только для покупки простых башен.

Вместо него появились две отдельные валюты:

- `🎲 Очки слияния` — случайное слияние;
- `🧬 Очки суперслияния` — ручной выбор двух башен и повышение уровня.

## Кубики судьбы

В каждом блоке из пяти волн игра скрыто рассчитывает от 3 до 10 кубиков.

Примеры блоков:

- волны 1–5;
- волны 6–10;
- волны 11–15;
- волны 16–20;
- волны 21–25;
- волны 26–30.

Кубики распределяются случайно внутри своего блока. Их количество и точная волна появления не показываются в прогнозе.

Каждый кубик:

- белого цвета;
- использует грань `⚀`, `⚁`, `⚂`, `⚃`, `⚄` или `⚅`;
- имеет значение от 1 до 6;
- становится быстрее и получает больше HP при высокой грани;
- не наносит урон базе;
- не даёт обычную энергию;
- после уничтожения приносит столько очков слияния, сколько точек на грани.

## Случайное слияние

Стоимость: `1 🎲`.

Правила:

- игра сама выбирает две установленные башни одного уровня;
- можно соединять любые цвета, включая две башни с одинаковым доминирующим цветом;
- цветовой результат определяется прежней таблицей вероятностей 5/5–9/1;
- уровень результата остаётся прежним;
- одна из двух позиций сохраняется, вторая башня исчезает;
- энергия не списывается.

## Суперслияние

Мини-босс и большой босс после уничтожения дают по `1 🧬`.

Правила:

- игрок вручную выбирает первую башню;
- после включения режима допустимые цели отмечаются значком `🧬`;
- второй башней может быть любая башня того же уровня;
- результат получает следующий уровень;
- уровень 2 даёт +50% к базовым характеристикам;
- уровень 3 даёт +100%;
- каждый следующий уровень добавляет ещё 50%;
- урон и радиус растут, перезарядка сокращается, количество целей может увеличиваться;
- цветовой результат всё равно определяется случайно по прежним шансам.

## Интерфейс

Добавлено:

- две валюты в верхней панели;
- компактный кошелёк слияний справа;
- кнопка случайного слияния;
- отдельная панель суперслияния;
- мигающий значок `🧬` над допустимыми целями;
- точный HP под каждым врагом;
- белое оформление кубиков;
- кнопки `+10 🎲` и `+3 🧬` в debug-режиме;
- валюты в итоговом отчёте партии.

## Честность случайности

Скрытым остаётся только количество и момент появления кубиков. Основные враги, боссы, HP-множители и скорость волны по-прежнему показываются заранее.

Seed карты определяет и форму маршрута, и скрытое распределение кубиков. Повтор того же seed воспроизводит ту же карту и тот же кубический сюрприз.

## Проверка

```bash
npm ci
npm run build
npm run lint
npm run dev
```

Для быстрого теста открыть:

```text
http://localhost:5173/?debug=1
```

И проверить:

1. `+10 🎲` позволяет выполнить случайное слияние двух башен одного уровня.
2. `+3 🧬` позволяет выбрать две башни одного уровня вручную.
3. После суперслияния уровень повышается.
4. На подходящих целях появляется значок `🧬`.
5. Кубики отображаются белыми гранями `⚀–⚅`.
6. Убитый кубик добавляет очки по значению грани.
7. Убитый босс добавляет одно очко суперслияния.
8. Энергия не списывается за слияния.
9. HP каждого врага видно под его символом.
