# v0.5.0 — Цветовой Завет

## Слоган

**Три цвета. Двадцать семь гибридов. Один путь, который нужно защитить.**

## Главная перемена

Обычная кнопка улучшения убрана. Теперь башни развиваются только через цветовое слияние.

Игрок выбирает первую башню, включает режим слияния и нажимает на подсвеченную совместимую башню. Цена каждого варианта показывается прямо на поле до совершения действия.

## Что добавлено

- красный, зелёный и синий цветовой состав;
- 27 первых двухцветных результатов;
- вероятности от 70% для 5/5 до 0.5% для конкретного 9/1;
- уникальные названия гибридов;
- точные характеристики и способности;
- стоимость следующего слияния, зависящая от редкости;
- пошаговая подсказка слияния;
- зелёная индикация доступной цены;
- красная индикация нехватки энергии;
- атлас слияний в отдельной боковой панели без прокрутки всей игры;
- 10 волн вместо 5, чтобы игрок успевал строить цепочки слияний.

## Правило способностей

- состав 5/5 активирует по одной способности каждого цвета;
- составы 6/4, 7/3, 8/2 и 9/1 активируют только способность доминирующего цвета;
- меньший цвет всё равно влияет на итоговые характеристики башни;
- редкий 9/1 получает максимальное влияние доминирующего цвета и легендарную способность.

## Что показывает атлас

Для каждого из 27 вариантов доступны:

- название башни;
- точный цветовой состав;
- вероятность выпадения;
- редкость;
- урон;
- радиус;
- время перезарядки;
- количество целей;
- полное описание способности;
- точное числовое действие способности.

Числа в атласе показывают результат первого слияния двух чистых башен — ранг 2.

## Ограничения версии

- пока реализованы только двухцветные башни;
- трёхцветные сочетания будут проектироваться после проверки баланса;
- визуальные снаряды, лазеры и импульсы появятся в отдельном патче;
- значения урона и сложности волн ещё требуют игровой балансировки.
